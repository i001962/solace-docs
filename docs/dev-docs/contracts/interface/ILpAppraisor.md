


## Functions
### appraise
```solidity
  function appraise(
    uint256 _tokenId
  ) external returns (uint256 _value)
```
Appraise a Uniswap LP Token.


#### Parameters:
| Name | Type | Description                                                          |
| :--- | :--- | :------------------------------------------------------------------- |
|`_tokenId` | uint256 | The id of the token to appraise.

#### Return Values:
| Name                           | Type          | Description                                                                  |
| :----------------------------- | :------------ | :--------------------------------------------------------------------------- |
|`_value`| uint256 | The token's value.
