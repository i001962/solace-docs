## Props

### `AccordionProps`
| Name | Type | Description                                                          |
| :--- | :--- | :------------------------------------------------------------------- |
| `isOpen` | boolean | Toggle visibility of accordion contents.
| `noscroll` | boolean | Disable scrollability.
